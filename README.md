# Supply Chain Order Analysis Dashboard
!(dashboard.png)
This project analyzes supply chain order data using Power BI.
## Features
- KPI metrics for Total Orders, Open Orders, and Back Orders
- Inventory distribution analysis by warehouse
- Product demand visualization
- Order trend analysis over time
- Interactive filters for warehouse and product
## Tools Used
- Power BI
- Excel
- Data Visualization
## Dashboard Insights
This dashboard helps identify supply chain bottlenecks, monitor order status, and analyze product demand patterns.
